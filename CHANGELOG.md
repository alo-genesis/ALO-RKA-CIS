# CHANGELOG

## v5.2 — 2026年3月

- 第2章に「2.0節（RKA=自律性 / CIS=社会性）」を新設し、アーキテクチャの二元論的本質を定義
- 第2章に「2.1節（3つの不変性の定義）」を新設し、Role不変性・Invariant・Constraintsの責務を厳密に分離
- 各構成要素（[R]〜[S]）の解説に「定義」「設計条件」「概念メタファー」の項目を導入し、規範的仕様書としての解像度を向上
- 基盤理論を「29」から「30」へ更新（不確実性管理の根幹として「ベイズ推論」を新設）
- 基盤理論の分類を「横断的基盤理論（9理論）」と「要素固有理論（21理論）」に再編（代数学・型理論・二次サイバネティクス・数理論理学を要素固有へ移動）
- 「オントロジー」をアーキテクチャ全体の存在根拠となる根源的横断理論として再定義
- 第3章タイトルを「構造・再帰・制御」へ、第4章タイトルを「多階層オブジェクトモデル」へ修正
- エグゼクティブ・サマリーへv5.2の主要な変更概念を追記
- [I].Invariantの設計条件名「区別性2」を「独立性」へ修正するなど、全体の推敲と表現の厳密化を実施
